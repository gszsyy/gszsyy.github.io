# 内容审核发布门禁

本仓库中的项目进度报告必须经过人工审核后才能推送到 GitHub Pages。

## 当前规则

- 未审核内容只能作为本地草稿存在。
- 页面、PPTX、图表和项目卡片内容都属于审核范围。
- 审核重点包括：事实准确性、措辞专业性、是否泄露不应公开的信息、是否误用本地参考资料。
- 原始参考文件不能直接挂载为正式内容；必须重构为经过审核的报告文本、图表或附件。

## 发布前检查

发布前至少确认：

- 报告正文已由负责人审核。
- PPT 下载文件与网页内容一致。
- 网页中没有未确认结论、临时措辞或内部备注。
- 没有直接提交大体积原始 PPT、原始视频或未授权素材。

审核确认后，才允许提交并推送到 `gszsyy.github.io`。
